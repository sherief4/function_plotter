# Function Plotter Application 
A simple application built with python that plots arbitary user entered functions using some python libraries such as numpy to generate the lists of x values and y values based on the function and the min x and max x which are given by the user, then show a plot using [Matplotlib](https://matplotlib.org/stable/index.html) and [Pyside2](https://pypi.org/project/PySide2/) in a simple graphical user interface using [Pytest-qt](https://pypi.org/project/pytest-qt/).
